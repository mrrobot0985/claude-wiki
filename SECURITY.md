# Security Policy

## Supported Versions

The following versions of `claude-wiki` receive security updates:

| Version | Supported |
| ------- | --------- |
| 0.8.x   | Yes       |
| < 0.8   | No        |

## Reporting a Vulnerability

Please report security vulnerabilities privately rather than opening a public issue.

- **Preferred:** Open a GitHub Security Advisory at [https://github.com/mrrobot0985/claude-wiki/security/advisories/new](https://github.com/mrrobot0985/claude-wiki/security/advisories/new)
- **Alternative:** Email the maintainer at [mrrobot0985@users.noreply.github.com](mailto:mrrobot0985@users.noreply.github.com)

Include a clear description of the issue, the steps to reproduce it, and the affected version. You can expect an initial response within five business days, and we will coordinate disclosure and remediation before any public discussion.

## Scope

Reports should relate to the `claude-wiki` Python package, its CLI, or its Claude Code hook handlers. Vulnerabilities in third-party dependencies should be reported to the upstream project, but you may notify us as well so we can update pinned requirements.
